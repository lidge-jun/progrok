[개념글 예약] SuperGrok / X Premium 결제해두고 API key 아깝던 놈들 필독 (Grok OAuth 프록시 progrok 배포)

형들, X Premium이나 SuperGrok 매달 구독료 내면서 브라우저 웹 UI로 노가다만 뛰고 있었냐?
API key 발급받자니 비용 이중지출이라 돈 아까웠던 갤러들을 위해 Grok OAuth 프록시 progrok 만들어서 배포한다.

Hermes Agent나 OpenClaw 쓰는 빌런들은 알겠지만, 걔네가 쓰는 xAI 공용 OAuth client ID (b1a00...) 기반 세션을 그대로 로컬 API로 바인딩해주는 도구임.

1. 이게 대체 뭐하는 물건임?
- 내 계정 세션 그대로 사용: OAuth PKCE / Device Code 로그인 한 번 해두면 로컬에 세션 저장되고 자동 갱신됨.
- OpenAI 호환 프록시: 로컬 포트 127.0.0.1:18645에 프록시 서버 띄워줌. SDK나 에이전트, Cursor, OpenClaw에서 API 주소만 로컬로 바꾸고 API key는 아무거나 넣으면 작동함.
- CLI & Web UI 탑재: 머신러닝/에이전트 연동용 CLI 커맨드는 물론이고, progrok chat 치면 로컬 웹 챗 UI까지 바로 열림.

2. xAI 공식 API 스펙(계약 표면) 완벽 대응
이 프록시가 대단한 이유는 단순히 텍스트만 전달하는 게 아니라, 최근 공식 xAI 오딧(Audit) 거친 API 계약 표면(Contract Surface)을 전부 로컬 환경에 완벽하게 바인딩해준다는 거다.

[TTS (Text-to-Speech) 계약]
- 지원 보이스: eve, ara, leo, rex, sal 5종 엔진 완벽 대응.
- output_format 객체 파싱 및 speed 세밀화 제어 (0.7배속부터 1.5배속까지 지원).
- 커스텀 보이스 생성: 단일 오디오 파일(최대 120초) 업로드 대응. 단, 일리노이주(BIPA) 제외 미국 지역 제한 및 기업용 API 권한 필요.

[STT (Speech-to-Text) 계약]
- 파일 직접 업로드 및 원격 URL 다운로드 방식 모두 지원.
- 파일 용량 한계: 최대 500MB 초대형 오디오 대응.
- 파일 우선순위(file-last) 매핑, 다이어라이즈(화자 분리), 멀티채널 오디오, keyterm 정밀 필터, raw audio 로데이터 바인딩 지원.

[Files API & 스토리지]
- 안정적인 파일 핸들링을 위해 보수적인 48MB 용량 제한 강제 적용 (공식 스펙 50MB 경계 충돌 방지).
- expires_after 타임스탬프 순서 제어 기능 탑재.
- 파일 내용 직접 조회(content) 및 조각 데이터(chunk) 스트리밍 엔드포인트 완벽 포팅.
- 문서 기반 대화 지원을 위한 attachment_search 및 벡터 검색 엔진 연동.

[Imagine Video 1.5 & V2V 비디오 생성 계약]
- 텍스트 투 비디오(T2V) 및 정지 이미지 애니메이팅(I2V).
- 레퍼런스 가이드 생성: 최대 7개의 레퍼런스 이미지를 묶어 최장 10초 분량 비디오 생성 가능.
- 비디오 편집(Edit) 및 연장(Extend): progrok video edit --video <url> 및 progrok video extend --video <url> 커맨드로 실시간 비디오 편집(grok-imagine-video 엔진 전용).

[검색 & 추론엔진 (Reasoning)]
- Responses API 전용 web_search 및 x_search 실시간 도구 기본 내장.
- 추론 강도 세밀 조절 지원 (none, low, medium, high, xhigh 5단계 reasoning effort 파싱).

[비용 & 쿼터 제어]
- 캐시 텍스트 할인 비율(cached text), Imagine 토큰 소모 계산, 보이스 전송 단가 및 도구 호출 비용 로깅 연동.

3. 주요 기능 및 커맨드
귀찮게 웹 UI 갈 필요 없이 터미널에서 다 됨:
- progrok login : 브라우저 켜지면서 깔끔하게 OAuth 로그인. SSH/원격 서버용 --device-code도 됨.
- progrok proxy : OpenAI 호환 서버 작동 시작.
- progrok search "<검색어>" : 웹 검색이랑 실시간 X 검색(citations 포함) 결과 바로 긁어옴.
- progrok image "<프롬프트>" : Imagine 기반 이미지 생성 및 이미지 에디팅.
- progrok video "<프롬프트>" : 이번에 새로 나온 Grok Video 1.5 비동기 생성 및 다운로드까지 터미널에서 올클리어.

⚠️ 4. 뇌절 금지 및 핵심 경고 (매우 중요)
사용하기 전에 이거 안 읽고 나중에 계정 터졌다고 징징대면 국물도 없다.

- 비공식 우회 프록시 리스크: 이 도구는 공식 API key 발급 경로가 아니라, Hermes Agent나 공식 앱이 사용하는 OAuth 인증 라인을 로컬 프록시 형태로 연동하는 거다.
- 언제든 터질 수 있음: Hermes에서 공식 지원하지 않거나 웹 UI 전용 비공식 표면(특히 이미지/비디오 우회 생성, 실시간 STT 웹소켓 등)은 xAI 측에서 패치하면 언제든지 예고 없이 기능이 동작하지 않거나 터질 수 있음. (참고: 실시간 WebSocket 엔드포인트는 프록싱에서 제외됨)
- 계정 정지 및 Suspension 책임은 본인에게: 비공식 API를 통한 과도한 호출, 봇 탐지 정책 위반 등으로 인해 발생할 수 있는 X/xAI 계정 제한, 정지(Suspension), 구독 해지 등의 모든 책임은 100% 사용자 본인에게 있다. 쫄리면 정식 API 요금제 결제해서 써라. 리스크 테이킹할 놈들만 쓰셈.
- 로컬 보안 주의: 기본적으로 로컬호스트(127.0.0.1)에만 바인딩되니까 포트 외부로 그냥 열지 마라. 세션 털리면 네 계정으로 결제되거나 털린다.

5. 어떻게 씀? (설치 및 실행)
Node.js 18 이상만 있으면 글로벌 설치 한 방에 끝난다.

npm install -g progrok
progrok login
progrok proxy
progrok search "Node.js 22 신기능" --web
